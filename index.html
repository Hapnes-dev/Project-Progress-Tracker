Rocketlane import
Manual capture (if Auto-fill doesn't work)
  1. Open kiona.rocketlane.com while logged in.
  2. Press F12 → Console tab, run: 
    JSON.parse(localStorage.__api_key)
    Returns ["api_key", "<uuid>", <userId>, <accountId>].
  3. Copy the <uuid> (second item) — paste it into the Session key field above.
  4. Click Save.
The key rotates occasionally; revisit Rocketlane (with the Tampermonkey bridge installed) to refresh automatically, or repeat these steps.
